How to sent a friend request ?

If the account you're looking to add is someone you know, the easiest way is to message them a Steam-generated "Friend Code."

1. Open the Steam application and select "Friends & Chat" from at the bottom right of the app (or from the menu bar at the top of your screen on a Mac).

2. Click it and it will appear a friend list. list.

3.Then click "Add a friend" button.

4.After you click "Add a friend " button, it will appear a new tab with your code and a space to write you'r friends code to add them. 5. After you write down you'r friend's code then press the button "Enter" from your keyboard.

They'll get a notification that you've sent a friend request. When they accept the invite, you'll be added to each other's Friends List.
